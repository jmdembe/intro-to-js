Introduction to JavaScript

Open up this directory in your code editor and then open the "code.js" file in the "js" directory. There are specific tasks in that file that you must complete. Once you have written your code, return to this page, refresh the page, then view any errors or other messages in the "console" in Chrome's developer tools (you can open that with "option+command+i"). Do not edit anything at the bottom of that file, that is code to ensure the code you wrote works (the tests)!

  2. Save that file and refresh this page
